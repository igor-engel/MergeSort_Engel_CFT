Версия Java 1.8.0_265
Сборка Maven

1. Открываем командную строку.
2. Переходим в директорию с файлом sort-it.jar.
3. Набираем команду java -jar sort-it.jar.
4. Следуем инструкции на экране.

Программа заточена под работу с любым количеством файлов более одного.
При наличии ошибок в файлах, часть данных отбрасывается.
Результат выйдет согласно найденным файлам.
